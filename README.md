# Django Banking REST API

This project is a Banking Backend Application developed using Django and Django REST Framework (DRF).

The application provides REST APIs for managing bank accounts, ledger entries, and money transfers between users.

## Features
- Account management
- Balance tracking
- Debit and credit ledger entries
- Money transfer system
- REST API architecture
- SQLite database integration
- API testing support using Postman

## Technologies Used
- Python
- Django
- Django REST Framework
- SQLite

## Project Purpose
This project was created for learning and practice purposes to improve backend development skills using Django and REST APIs.
